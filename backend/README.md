# MEDCARE - Hospital Management System

A premium, modern Hospital Management System (HMS) designed for efficient healthcare administration and patient care.

## 🚀 Live Demo
Check out the live application here: **[https://pseftis.github.io/hms/](https://pseftis.github.io/hms/)**

## ✨ Features
- **Modern Dashboard**: Real-time health metrics and interactive visuals.
- **Hospital Network**: Search and explore top-rated medical facilities.
- **Expert Doctors**: Meet and book appointments with world-class specialists.
- **Seamless Appointments**: Dynamic scheduling and management system.
- **Premium UI/UX**: Built with Framer Motion, Material UI, and Glassmorphism design principles.

## 🛠️ Tech Stack
- **Frontend**: React.js, Material UI, Framer Motion, Lucide Icons.
- **Backend API**: Integrated Demo Data & RESTful services.
- **Styles**: Custom Glassmorphism and Responsive Layouts.
